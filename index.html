FitBalance

Última actualización: 7 de abril de 2026

FitBalance es una aplicación móvil diseñada para registrar alimentos, analizar información nutricional y visualizar estadísticas personales. Tu privacidad es fundamental para nosotros.

1. Información que recopilamos

FitBalance puede recopilar la siguiente información:

  • Datos introducidos por el usuario (alimentos, comidas, registros diarios).
  • Información generada por el uso de la app (historial de comidas).
  • Datos obtenidos de APIs externas (por ejemplo, información nutricional de bases de datos de alimentos).

2. Uso de la información

La información recopilada se utiliza exclusivamente para:

  • Mostrar estadísticas nutricionales detalladas.
  • Registrar y visualizar el progreso del usuario.
  • Mejorar la experiencia de uso y optimizar las funciones de la app.

3. Almacenamiento de datos

  • Los datos se almacenan localmente en el dispositivo del usuario.
  • No se almacenan en servidores propios (a menos que se indique lo contrario en futuras versiones mediante una actualización de esta política).

4. Compartición de datos

  • FitBalance no comparte datos personales con terceros.
  • Algunas búsquedas pueden consultar APIs externas para obtener información nutricional, pero estas consultas no vinculan tu identidad personal.

5. Permisos

La aplicación puede solicitar acceso a:

  • Cámara: Únicamente para escanear códigos de barras de alimentos y agilizar el registro.

6. Seguridad

Se toman medidas razonables y estándares de la industria para proteger la información almacenada en tu dispositivo.

7. Cambios en la política

Esta política puede actualizarse en futuras versiones para reflejar nuevas funcionalidades o cambios legales. Te recomendamos revisarla periódicamente.
